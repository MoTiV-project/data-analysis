# MoTiV: Mobility and Time Value

This repository contains the code used for the data analysis collected during
the MoTiV project.

## The MoTiV project

![MoTiV project logo](https://motivproject.eu/typo3conf/ext/wb_template/i/motiv_logo.png)

The Mobility and Time Value "MoTiV" project addresses emerging needs and
perspectives on Value of Travel Time (VTT), a relevant research area
particularly valuable to decision-makers, transportation planners, engineers,
and economists in the context of projects aiming at enhancing transportation
infrastructure.

For further info visit the project page: [motivproject.eu][motiv].

### Goals

1. Introduce and validate a conceptual framework for the estimation of Value
of Travel Time (VTT)​
2. Explore the dynamics of individual preferences, behaviors and lifestyles
that influence travel and mobility choices​
3. Address VTT from the perspective of personality, preferences, needs and
expectations of the single individual, in contrast with the traditional
viewpoint of the economic dimension (time and cost savings).​
4. Contribute to advance research on VTT by introducing a conceptual framework
for the estimation of VTT at an individual level based
